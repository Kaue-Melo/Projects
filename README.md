# Projects
Criei esse script para agilizar o trabalho manual com conferência da escrita em descrição de produtos recebidos por terceiro. A ideia é que ele passa por diversas palavras sortidas em um dicionário privado e corrigiria casos como:

acai > Açaí
ped > Pedaço

Uma forma 'arcaica' considerando que precisa haver um dicionario que identifique o termo errado e corrigisse, mas que com pouco tempo se mostrou MUITO eficaz no dia a dia de trablaho da equipe.



Tecnologia:
- pandas
- re
